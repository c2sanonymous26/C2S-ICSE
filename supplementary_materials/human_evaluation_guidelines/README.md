# Human Evaluation Guidelines

This directory contains the supplementary materials used for the RQ1 human evaluation of constraint quality.

## Contents

- `guide.md`: the evaluator-facing instructions. It defines the two evaluation questions and the Level A/B/C criteria.
- `example.md`: example constraints with reference explanations for Level A, Level B, and Level C.
- `background/`: background materials for the taxi scenario.
  - `scenario.md`: the target application scenario.
  - `fields.md`: the context-field descriptions.
- `questionnaire/`: blank evaluation forms used for assigning Level A/B/C labels.
  - `group_1.md`
  - `group_2.md`

## Usage

Evaluators should read the files in the following order:

1. `guide.md`
2. `background/scenario.md` and `background/fields.md`
3. `example.md`
4. the assigned questionnaire file in `questionnaire/`

The corresponding constraint files used in the evaluation are included in:

```text
../../evaluation_results/results/RQ1/effectiveness_humanEvaluation/unique_constraints/
```
